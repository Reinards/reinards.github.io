Our first game May 25, 2017

Hello again!

The Squane is out!

The development of this game was inactive for more than a year. But I did it.
Niks wouldn’t want to participate(kinda), because I used the Godot game engine to rebuild this game.\

At first, the rebuild was not good. It had slow performance, it was too easy and repeating.
After publishing it, there were some recommendations to make this game more difficult, what I did.
I added mini-map for colors, I made slide-selecting possible, I also improved performance and data saving mechanics.

The game didn’t get more than 50 downloads, but it was our first game. We have earned 5 cents in total, haha.

My next game will be better. I cannot think of something interesting and fun, but I know the idea will come and as soon as it happens, I will try to realise it, just for fun.

If anyone wonders about what is Godot, then I can say it is the best game engine I have found so far!

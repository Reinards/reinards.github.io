Welcome to my blog! March 26, 2017

Welcome!

So, I am 17 years old boy from Latvia. I have been interested in programming since 2012.
My friend Niks, was already learning programming for one year when he tried to convince me to try it too. I started with competitive programming - the only thing I was/am taught by that teacher.

I have always been interested in learning things that are useful in the real life. I cannot remember when it happened, but the same friend who interested me in competitive programming taught me a little bit of html and css.

I can still remember that green website which used Javascript to toggle a div :D

That spark started a fire in me. From then, more or less, I have been learning html, css, javascript and many other programming things.
Each year there was a programmers camp in which we were taught different technologies. PHP was a very cool one. After that one camp I had an intention to make a world-changing website… like Facebook.
Of course, I didn’t make anything big.

For the last one year I have been interested even more.
I have learned to use SASS (It is so much easier to write css now).
I have learned to make simple websites with Django and Jekyll (This website is powered by Jekyll :P)
I have discovered many awesome frameworks and libraries.
I have learned a bunch of stuff.

I made this website because I feel the need for a place where I can express my thoughts and show off my work.
In a nutshell, I will make a post whenever I have something cool to say to you.

In future, I am planning to develop a website + mobile app (Secret). I also have some ideas for desktop games that I could make.
So, stay tuned and have a nice day!
